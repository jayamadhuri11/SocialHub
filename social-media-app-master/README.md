# Social Hub

## Overview

Social Hub is a modern social media platform that allows users to connect, share posts, interact through comments, like content, and follow other users. The platform provides a simple and engaging environment for online communication and community building.

## Features

- User Authentication (Login & Signup)
  
- User Profiles
  
- Create, Edit, and Delete Posts
  
- Comments on Posts
  
- Like System
  
- Follow/Unfollow Users
  
- Search Functionality
  
- Responsive Design
  
- Real-Time User Interaction

## Technologies Used

### Frontend

- React.js
  
- HTML
  
- CSS
  
- JavaScript
  
- Material UI

### Backend

- Node.js
  
- Express.js

### Database

- MongoDB

## Installation

### Install Backend Dependencies

npm install

### Start Backend Server

node server.js

### Install Frontend Dependencies

cd client

npm install

### Start Frontend

npm start

## Usage

1. Create an account or log in.

2. Create and share posts.

3. Like and comment on posts.

4. Follow other users.

5. Explore and interact with community content.
